# Readme

# Data and R Scripts for Bachelor Thesis

This repository contains the data and R scripts used for the analyses in the bachelor thesis:
  
  "Behaviorale Adaption in virtueller Realität: Zusammenhänge von Performanz, Cybersickness und subjektivem Spielerleben"

## Contents
- /data: anonymized datasets
- /scripts: R scripts for preprocessing and statistical analyses
- /results: derived results and figures

## Software
- R version 4.5.1

## Notes
Raw data containing personal information were not uploaded due to data protection regulations.
